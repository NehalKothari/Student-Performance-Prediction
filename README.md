 Student Performance Predictor
 Overview

This project is a Machine Learning-based web application that predicts student final exam performance based on behavioral and academic features.

Features

* Predict student marks using ML
* Decision Tree Regression model
*  Feature-based prediction
* Interactive Streamlit UI

Technologies Used

* Python
* Pandas
* Scikit-learn
* Streamlit

 How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

## ML Approach

* Data preprocessing and feature selection
* Model training using Decision Tree Regressor
* Performance evaluation using R² Score

## Insights

* Internal marks (G1, G2) strongly influence final marks
* Behavioral features have moderate impact

## Use Case

* Academic performance prediction
* Educational analytics
